# Central library for Embedded software of Black Pearl Racing

## PlatformIO project structure

```
project structure
   |proj
   |   |src
   |	 |  |__main.cpp
   |	 |
   |	 |lib
   |      |<lib_name>
   |     	          |__<lib\_name>.cpp
   |		            |__ <lib_name>.h
```


## How to import this library into your Embedded Project

1. Clone this project or Download .zip
2. Then select the folder that you want from the library
3. Copy it into your Platformio project `<your_proj>/lib/.`
